## Veo3 Prompt Style Guide

All prompts follow this consistent aesthetic:

### Visual Style
- **Aesthetic:** Professional, modern, tech/SaaS
- **Camera Work:** Smooth movements, cinematic
- **Composition:** Clean, minimal, high production value
- **Color Palette:** Purple and blue gradients (brand colors)
- **Lighting:** Professional, natural when appropriate
- **Quality:** 4K resolution, detailed textures

### Prompt Structure Template
```
[Shot type and movement], [subject/action description],
[environment/background], [color palette], [aesthetic reference],
[technical quality notes], [resolution specification]
```

### Example Prompt
```
Cinematic opening shot, smooth camera push forward through minimalist
modern office space with floating holographic design elements in purple
and blue gradient colors, clean white environment with natural lighting,
professional tech startup aesthetic, high production value, shallow depth
of field, 4K ultra detailed, modern SaaS brand reveal
```

## Technical Specifications Summary

### Video Export Standards

**High-Quality (Landing Page, Showcase)**
- Format: H.264, MP4
- Resolution: 4K (3840x2160)
- Frame Rate: 60fps
- Bitrate: 20-25 Mbps
- Color Space: Rec. 709
- Audio: AAC, 320kbps, stereo

**Standard Quality (Explainers, Demos)**
- Format: H.264, MP4
- Resolution: Full HD (1920x1080)
- Frame Rate: 30fps
- Bitrate: 12-15 Mbps
- Color Space: Rec. 709
- Audio: AAC, 256kbps, stereo

**Email-Optimized**
- Format: H.264, MP4
- Resolution: 1920x1080
- Frame Rate: 30fps
- Bitrate: 2-3 Mbps
- File Size: <1MB
- Fallback: Animated GIF

**Social Media (Vertical)**
- Format: H.264, MP4
- Resolution: 1080x1920 (9:16)
- Frame Rate: 30fps
- Bitrate: 8 Mbps
- File Size: <30MB
