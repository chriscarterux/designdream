## Production Workflow

### Step 1: Scene Generation with Veo3 API
1. Authenticate with Veo3 API
2. Submit Scene 1 prompt → generates 5-second clip
3. Submit Scene 2 prompt → generates 7-second clip
4. Submit Scene 3 prompt → generates 10-second clip
5. Submit Scene 4 prompt → generates 8-second clip
6. Download all scene files in 4K resolution

### Step 2: Audio Production
1. Record professional voiceover
2. Generate or source background music (electronic/corporate style)
3. Record or download sound effects
4. Mix audio in DAW (Adobe Audition, Premiere Pro, or Logic Pro)
5. Export final audio track at 320kbps AAC

### Step 3: Video Editing
1. Import all 4 Veo3-generated clips into editor
2. Arrange clips in sequence with proper timings
3. Add text overlays with animations:
   - Fade in/out transitions
   - Scale and position animations
   - Font: Modern sans-serif (e.g., Inter, Sohne)
   - Color: White with subtle shadows
4. Color grade for consistency:
   - Ensure purple/blue color grade is uniform
   - Adjust exposure and contrast
   - Add film grain for premium look (light, 2-5%)
5. Sync voiceover and music
6. Add sound effects at key moments

### Step 4: Composite & Effects
1. Add subtle particle effects or glows (if needed)
2. Apply smooth transitions between scenes
3. Add final color grading pass
4. Export master file at highest quality

### Step 5: Export & Optimization
1. **Desktop version (1920x1080):**
   - H.264 codec
   - 20 Mbps bitrate
   - File size: ~75-80 MB

2. **4K version (3840x2160):**
   - H.264 codec
   - 30+ Mbps bitrate
   - File size: ~250-300 MB

3. **Social media version (1080x1080):**
   - Crop from 16:9 original
   - Add subtle vignette/fade to edges
   - 15 Mbps bitrate
   - File size: ~30-40 MB

4. **Compressed web version:**
   - 8 Mbps bitrate for fast loading
   - File size: ~20-25 MB

### Step 6: Generate Subtitles
- Create .srt file with all text overlays and voiceover as captions
- Timing must match video exactly
- Include in final deliverables
